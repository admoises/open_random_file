# open_random_file
Script para abertura aleatória de arquivo com determinada extensão em repositórios pré-definidos. O script contém dois métodos para abertura. O primeiro onde se seleciona o repositório (disco) e o outro que se decide totalmente sem intervenção do usuário.

# Dependências:
  pip install python-dotenv
  ## https://pypi.org/project/python-dotenv/
